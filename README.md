# Frogger

An iteration of the classic arcade game Frogger.

Guide your frog across busy roads and treacherous rivers to reach the goal.
Dodge cars, hop on logs and turtles, and don't run out of time — or lives.

**[Play in your browser](https://angelgalindo7.github.io/Frogger/)**

![Gameplay screenshot](images/image.png)

## Controls
- **W / ↑** — move up
- **S / ↓** — move down
- **A / ←** — move left
- **D / →** — move right
